<p><strong>Océan</strong>: Un Titan puissant qui avec sa sœur engendre près de 3000 <a title="Océanides" href="http://www.genealogiedesdieuxgrecs.com/Genealogie_des_Dieux_grecs/category/les-divinites-grecques/nymphes/oceanide/">Océanides</a>. Il est le fils de <a title="Gaïa " href="http://www.genealogiedesdieuxgrecs.com/Genealogie_des_Dieux_grecs/2011/03/gaia/">Gaïa </a>et <a title="Ouranos " href="http://www.genealogiedesdieuxgrecs.com/Genealogie_des_Dieux_grecs/2011/03/ouranos/">Ouranos </a>et se tape une de ses <a title="Thétys" href="http://www.genealogiedesdieuxgrecs.com/Genealogie_des_Dieux_grecs/2011/03/thetys/">sœurs</a>, trait de caractère de cette famille, pour engendrer ces <a title="Océanides" href="http://www.genealogiedesdieuxgrecs.com/Genealogie_des_Dieux_grecs/category/les-divinites-grecques/nymphes/oceanide/">océanides</a>, dont trois sont représentées ici :
<ul>
 	<li><a title="Métis" href="http://www.genealogiedesdieuxgrecs.com/Genealogie_des_Dieux_grecs/2011/03/metis/">Métis</a>,</li>
 	<li>Pléloné et</li>
 	<li><a title="Clyméné" href="http://www.genealogiedesdieuxgrecs.com/Genealogie_des_Dieux_grecs/2011/03/clymene/">Clyméné</a>.</li>
</ul>
